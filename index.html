HTML Formatting

  • Bold text:
    This is bold text.

  • Strong/Imporant text:
    This is important text

  • Italic text:
    This text is italic.

  • Emphasized text:
    This text is Emphasized.

  • Small text:
    This is some smaller text

  • Highlighted text:

    Do not forget to pay bills tomorrow.


  • Deleted text:

    My favorite color is red black.


  • Inserted text:

    My favorite color is red black


  • Subscripted text:

    This is subscripted text.

    Some examples: H20, H2SO4 etc.


  • Superscripted text:

    This is superscripted text.

    Some examples:
    (a-b)2 = (a+b)2-4ab, 2(a2+b2) = (a+b)2+(a-b)2 etc.
